# NPU - National Progress and Unity Party

Official website for the Horumar iyo Midnimo Qaran (NPU) political party.

## About

This is the official online presence of the National Progress and Unity Party, showcasing our leadership, platform, and vision for Somalia's future.

## Features

- Modern, responsive design with cyberpunk aesthetic
- Interactive leader profiles
- Comprehensive platform pillars
- Latest updates and news
- Contact and engagement options

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view the site locally.

## Deployment

This site is built with Vite and React and can be deployed to any static hosting platform.
